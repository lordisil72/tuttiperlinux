Bisogna prima di tutto capire che quando viene cancellato un file ¬†esso non viene eliminato fisicamente, piuttosto il filesystem lo “marca” come cancellato rendendo cosi’ disponibile alla scrittura di altri file la porzione di disco che questo occupava.   • sudo photorec

Appena avviato photorec ¬†vi chiedera’, nel seguente ordine, queste informazioni :
